# LangChain Search Chatbot 🚀

An AI-powered chatbot built with Streamlit, LangChain, and Groq that can search the web, retrieve Wikipedia information, and fetch Arxiv research papers. It uses LangChain agents to intelligently select tools and stream real-time responses through an interactive chat interface.

## Features ✨

* Web search using DuckDuckGo
* Wikipedia information retrieval
* Arxiv research paper search
* Real-time streaming responses
* Interactive chat interface
* Tool-based AI agent reasoning

## Tech Stack 🛠️

* Python
* Streamlit
* LangChain
* Groq API
* DuckDuckGo Search
* Wikipedia API
* Arxiv API

## Installation ⚙️

```bash
git clone <https://github.com/charrann12/SearchEngine.git>

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

## Run the Application ▶️

```bash
streamlit run app.py
```

## API Key 🔑

Get your Groq API key from:
https://console.groq.com

Enter the API key in the Streamlit sidebar before using the chatbot.
